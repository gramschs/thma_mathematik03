# Komplexe Eigenwerte und das Modell der gedämpften Schwingung

## Lernziele

```{admonition} Lernziele
:class: attention
* [ ] Sie können für $D < 0$ die beiden konjugiert komplexen Eigenwerte $\lambda_{1,2} = -\alpha \pm i\omega_0$ bestimmen und Real- und Imaginärteil benennen.
* [ ] Sie wissen, wie die **Euler-Formel** $e^{i\varphi} = \cos\varphi + i\sin\varphi$ genutzt wird, um komplexe Exponentialfunktionen in reelle Lösungen umzuschreiben.
* [ ] Sie können das **reelle Fundamentalsystem** $\{e^{-\alpha x}\cos(\omega_0 x),\, e^{-\alpha x}\sin(\omega_0 x)\}$ angeben und die allgemeine Lösung $y_h(x) = e^{-\alpha x}\bigl(C_1\cos(\omega_0 x) + C_2\sin(\omega_0 x)\bigr)$ aufschreiben.
* [ ] Sie können die Parameter $\alpha$ (Abklingrate) und $\omega_0$ (Eigenkreisfrequenz) physikalisch als **gedämpfte Schwingung** interpretieren.
```
