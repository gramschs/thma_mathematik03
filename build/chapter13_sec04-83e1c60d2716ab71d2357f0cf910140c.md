# 13.4 Überblick und Ausblick

## Lernziele

```{admonition} Lernziele
:class: attention
* [ ] Sie können für eine gegebene periodische Funktion die effizienteste
  Lösungsstrategie auswählen: allgemeine Euler-Fourier-Formeln oder Ausnutzung
  von Geraden- bzw. Ungeraden-Symmetrie.
* [ ] Sie wissen, dass es neben der reellen auch eine **komplexe Darstellung**
  der Fourierreihe gibt, und kennen deren Bedeutung für die Regelungstechnik
  und Signalverarbeitung.
* [ ] Sie kennen den konzeptionellen Schritt von der Fourierreihe zur
  **Fouriertransformation** und wissen, in welchen Lehrveranstaltungen Sie
  dieses Werkzeug vertiefen werden.
```
