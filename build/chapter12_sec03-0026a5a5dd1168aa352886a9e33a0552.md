# 12.3 Die Euler-Fourier-Formeln

## Lernziele

```{admonition} Lernziele
:class: attention
* [ ] Sie können die **Euler-Fourier-Formeln** für die Koeffizienten $a_0$,
  $a_n$ und $b_n$ korrekt aufschreiben.
* [ ] Sie können die Fourierkoeffizienten einer stückweise definierten
  periodischen Funktion durch Integration berechnen.
* [ ] Sie können die zugehörige **Fourierreihe** aufstellen und die ersten
  Partialsummen angeben.
* [ ] Sie können ein berechnetes Ergebnis auf Plausibilität prüfen, indem Sie
  Partialsummen an ausgewählten Stellen mit dem Funktionswert vergleichen.
```
