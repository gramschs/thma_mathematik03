# 12.2 Das trigonometrische Funktionssystem

## Lernziele

```{admonition} Lernziele
:class: attention
* [ ] Sie kennen die Funktionen $1$, $\cos(n\omega_0 t)$ und $\sin(n\omega_0 t)$
  als Bausteine der Fourierreihe und verstehen, warum genau diese Funktionen
  gewählt werden.
* [ ] Sie wissen, was **Orthogonalität** zweier Funktionen bedeutet, und können
  die Orthogonalitätsrelationen des trigonometrischen Systems benennen.
* [ ] Sie verstehen die Grundidee, wie die Orthogonalitätseigenschaft genutzt
  wird, um **Fourierkoeffizienten** aus einem gegebenen Signal herauszuprojizieren.
```
