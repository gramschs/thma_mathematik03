## Lernziele

```{admonition} Lernziele
:class: attention
* [ ] Sie können eine diagonalisierbare Matrix $\mathbf{A}$ effizient potenzieren,
  indem Sie die Formel
  $\mathbf{A}^k = \mathbf{V}\mathbf{D}^k\mathbf{V}^{-1}$
  anwenden.
* [ ] Sie wissen, dass $\mathbf{D}^k$ durch einfaches Potenzieren der
  Diagonalelemente berechnet wird:
  $\mathbf{D}^k = \text{diag}(\lambda_1^k, \ldots, \lambda_n^k)$.
* [ ] Sie verstehen den Effizienzgewinn der Diagonalisierung gegenüber der
  wiederholten Matrizenmultiplikation, besonders bei großen $k$.
```
