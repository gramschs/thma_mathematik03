# 13.3 Erzwungene Schwingung mit periodischer Erregung

## Lernziele

```{admonition} Lernziele
:class: attention
* [ ] Sie verstehen, wie das **Superpositionsprinzip** bei linearen ODEs erlaubt,
  eine periodisch erregte Schwingung komponentenweise zu lösen.
* [ ] Sie können die Grundidee der Lösung skizzieren: Fourierreihe der Erregung
  aufstellen, jede Komponente separat durch die ODE schicken, Ergebnisse
  überlagern.
* [ ] Sie kennen die Bedeutung von **Resonanz** im Zusammenhang mit der
  Fourierzerlegung und können erklären, welche Oberwelle eines Erregersignals
  ein System in Resonanz versetzen kann.
```
