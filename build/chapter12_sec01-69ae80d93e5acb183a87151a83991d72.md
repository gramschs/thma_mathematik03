# 12.1 Periodische Signale im Maschinenbau

## Lernziele

```{admonition} Lernziele
:class: attention
* [ ] Sie kennen den Begriff der **Periode** $T$ und der **Grundkreisfrequenz**
  $\omega_0 = 2\pi/T$ und können beide Größen aus einem periodischen Signal
  ablesen oder berechnen.
* [ ] Sie wissen, was **Oberwellen** sind, und können die Frequenzen der ersten
  Oberwellen zu einer gegebenen Grundfrequenz bestimmen.
* [ ] Sie verstehen, warum periodische Signale im Maschinenbau allgegenwärtig
  sind, und kennen mindestens zwei konkrete Beispiele aus der Praxis.
* [ ] Sie können die zentrale Frage der Fourieranalyse formulieren: Lässt sich
  ein beliebiges periodisches Signal als Überlagerung von Sinusschwingungen
  darstellen?
```
