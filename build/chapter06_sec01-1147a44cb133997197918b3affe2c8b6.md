# Was ist eine Differentialgleichung?

## Lernziele

```{admonition} Lernziele
:class: attention
* [ ] Sie wissen, was eine **gewöhnliche Differentialgleichung** (gDGL) ist, und können sie von einer partiellen Differentialgleichung unterscheiden.
* [ ] Sie kennen die Begriffe **Ordnung** und **Grad** einer gDGL und können diese an konkreten Beispielen bestimmen.
* [ ] Sie können zwischen einer **allgemeinen Lösung** und einer **speziellen (partikulären) Lösung** einer gDGL unterscheiden.
* [ ] Sie wissen, was ein **Anfangswertproblem (AWP)** und ein **Randwertproblem (RWP)** sind, und können für ein gegebenes AWP die Anfangsbedingungen benennen.
```
