# Das Euler-Verfahren: eine Lösungskurve Schritt für Schritt verfolgen

## Lernziele

```{admonition} Lernziele
:class: attention
* [ ] Sie können die Herleitung des **Polygonzugverfahrens von Euler** nachvollziehen, indem Sie den Differenzenquotienten als Näherung für die Ableitung einsetzen.
* [ ] Sie beherrschen das **explizite Euler-Verfahren** und können damit zu einem gegebenen AWP der Form $y' = F(x, y)$, $y(x_0) = y_0$ eine numerische Näherungslösung berechnen.
* [ ] Sie können die entstehende Punktfolge $(x_n, y_n)$ in einem Koordinatensystem als Polygonzug skizzieren.
* [ ] Sie kennen die **Schwächen des Euler-Verfahrens** (akkumulierende Fehler, Probleme bei stark gekrümmten Lösungskurven) und können diese an einem Beispiel erläutern.
```
