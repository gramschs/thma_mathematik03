# Lineare DGL 2. Ordnung: wenn auch die Beschleunigung eine Rolle spielt

```{admonition} Warnung
:class: warning
Dieses Vorlesungsskript befindet sich im Umbau.
```

## Lernziele

```{admonition} Lernziele
:class: attention
* [ ] Sie kennen die Definition einer **linearen gDGL 2. Ordnung mit konstanten Koeffizienten** der Form $y'' + ay' + by = g(x)$ mit $a, b \in \mathbb{R}$.
* [ ] Sie können zu einer gegebenen inhomogenen linearen DGL 2. Ordnung die **zugehörige homogene DGL** aufstellen.
* [ ] Sie wissen, dass die allgemeine Lösung der inhomogenen DGL 2. Ordnung ebenfalls die Struktur $y_{\text{allgemein}} = y_h + y_p$ hat, und dass $y_h$ nun zwei freie Konstanten $C_1$ und $C_2$ enthält.
* [ ] Sie können eine **partikuläre Lösung** durch Einsetzen der beiden Anfangswerte $y(x_0) = y_0$ und $y'(x_0) = v_0$ bestimmen.
```
