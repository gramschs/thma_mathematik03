# Separation der Variablen: die erste analytische Lösungsmethode

## Lernziele

```{admonition} Lernziele
:class: attention
* [ ] Sie erkennen eine **separierbare** (trennbare) gDGL 1. Ordnung der Form $y' = f(x) \cdot g(y)$.
* [ ] Sie beherrschen das Verfahren der **Separation der Variablen** und können die vier Lösungsschritte (Trennen, Integrieren, Stammfunktion einsetzen, nach $y$ auflösen) auf eine gegebene DGL anwenden.
* [ ] Sie können die **allgemeine Lösung** einer separierbaren gDGL angeben.
* [ ] Sie beachten den Sonderfall $g(y) = 0$ und können die zugehörige **konstante Lösung** bestimmen.
```
