## Lernziele

```{admonition} Lernziele
:class: attention
* [ ] Sie kennen die Definition **ähnlicher Matrizen**: Zwei Matrizen
  $\mathbf{A}, \mathbf{C} \in \mathbb{R}^{n\times n}$ heißen ähnlich, wenn es
  eine invertierbare Matrix $\mathbf{B}$ gibt, sodass
  $\mathbf{C} = \mathbf{B}^{-1}\mathbf{A}\mathbf{B}$.
* [ ] Sie wissen, dass ähnliche Matrizen dasselbe **charakteristische Polynom**
  und damit dieselben **Eigenwerte** besitzen.
* [ ] Sie verstehen, dass die Ähnlichkeitstransformation einem **Basiswechsel**
  entspricht: Die Abbildung $\mathbf{A}$ wird in der Basis der Spalten von
  $\mathbf{B}$ durch $\mathbf{C}$ dargestellt.
```
