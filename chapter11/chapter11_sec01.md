# Die partikuläre Lösung: Störfunktionen bei DGL 2. Ordnung

## Lernziele

```{admonition} Lernziele
:class: attention
* [ ] Sie können den **Ansatz vom Typ der rechten Seite** aus Kapitel 8 auf die inhomogene lineare DGL 2. Ordnung $y'' + ay' + by = g(x)$ übertragen und die unbekannten Koeffizienten durch Einsetzen und Koeffizientenvergleich bestimmen.
* [ ] Sie erkennen den **Resonanzfall** bei DGL 2. Ordnung: Wenn der Ansatz für $y_p$ eine Lösung der zugehörigen homogenen DGL ist, muss er mit $x$ multipliziert werden.
* [ ] Sie können die **allgemeine Lösung** $y_{\text{allgemein}} = y_h + y_p$ einer inhomogenen linearen DGL 2. Ordnung vollständig angeben.
```
