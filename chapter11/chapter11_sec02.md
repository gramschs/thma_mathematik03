# Anfangswertprobleme 2. Ordnung: zwei Bedingungen, zwei Konstanten

```{admonition} Warnung
:class: warning
Dieses Vorlesungsskript befindet sich im Umbau.
```

## Lernziele

```{admonition} Lernziele
:class: attention
* [ ] Sie wissen, dass ein Anfangswertproblem 2. Ordnung zwei Anfangsbedingungen $y(x_0) = y_0$ und $y'(x_0) = v_0$ erfordert, um die beiden Konstanten $C_1$ und $C_2$ eindeutig festzulegen.
* [ ] Sie können aus der allgemeinen Lösung $y_{\text{allgemein}} = C_1 y_1 + C_2 y_2 + y_p$ durch Einsetzen der Anfangsbedingungen ein **lineares Gleichungssystem** für $C_1$ und $C_2$ aufstellen und lösen.
* [ ] Sie können die vollständige **partikuläre Lösung** eines AWP 2. Ordnung berechnen und das Ergebnis durch Einsetzen der Anfangsbedingungen verifizieren.
```
