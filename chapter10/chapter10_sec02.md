# Wann spannen zwei Lösungen den gesamten Lösungsraum auf?

```{admonition} Warnung
:class: warning
Dieses Vorlesungsskript befindet sich im Umbau.
```

## Lernziele

```{admonition} Lernziele
:class: attention
* [ ] Sie wissen, was **Fundamentallösungen** $y_1$ und $y_2$ einer homogenen linearen gDGL 2. Ordnung sind, und was ein **Fundamentalsystem** $\{y_1, y_2\}$ auszeichnet.
* [ ] Sie können mit der **Wronski-Determinante**
  \begin{equation*}
  W(y_1, y_2) = \det \begin{pmatrix} y_1(x) & y_2(x) \\ y_1'(x) & y_2'(x) \end{pmatrix} = y_1 y_2' - y_2 y_1'
  \end{equation*}
  nachweisen, ob zwei Lösungen linear unabhängig sind und damit ein Fundamentalsystem bilden.
* [ ] Sie können aus einem Fundamentalsystem $\{y_1, y_2\}$ die **allgemeine Lösung** $y_h(x) = C_1 y_1(x) + C_2 y_2(x)$ der homogenen DGL angeben.
```
